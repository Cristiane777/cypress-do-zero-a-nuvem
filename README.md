# Cypress do zero a nuvem

Projeto exemplo de cypress 

## Pré requisitos 

Possuir o `Git`, `Node.js` e `NPM` instalados.

## Instalação

Rodar `npm install`

## Testes

Esse projeto você rodar em desktop ou mobile

## Desktop

Rodar `npm test` para rodar em modo headless
Ou, rodar `npm run cy:open` para abrir no cypress App

## Mobile

Rodar `npm run test:mobile` para rodar em modo headless
Ou, rodar `npm run cy:open:mobile` para abrir no cypress App
